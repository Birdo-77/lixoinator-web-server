# Lixoinator (ou recyclinator)

Este e um projeto escolar para o programa agrinho.

Sua ideia e ser um coletor de latinhas automatizado, onde o aluno coloca sua latinha na maquina e recebe um credito para gastar na cantina da escola

O servidor e feito com node js e pode ser rodado com:
node index.js

o codigo python pode ser executado na pasta pythonCode
python3 index.py

foi usado um raspberry pi 2 com dietpi integrado pelo USB com um arduino UNO.
